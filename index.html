SwitchBot Controller プレビュー

このページは Pull Request ごとに SwitchBot Controller の UI 挙動を Cloudflare Pages 上で確認するためのプレースホルダーです。Electron や SwitchBot API の実呼び出しは行われず、見た目やモック挙動の確認に専念できます。

目的

  • PR の変更をブラウザで手軽に共有・レビュー
  • API キー不要で UI の差分を確認
  • Cloudflare Pages のデプロイ動作を事前検証

確認ポイント

  • スタイルやコンポーネントの見た目
  • ルーティングや状態管理のモック挙動
  • 多言語対応やテーマ切り替えなどの UI

注意事項

  • SwitchBot Cloud API は呼び出しません（通信なし）
  • Electron 固有の機能はブラウザでは無効です
  • 必要に応じてモックデータを利用して動作を再現してください
最新版は PR ごとの Cloudflare Pages プレビューでご確認ください。
